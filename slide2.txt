新規問題作成AI（BtoB）発表原稿
発表時間：5分

===== スライド1：タイトル =====
皆さん、こんにちは。
今日は「新規問題作成AI『問ジェネ』」について発表させていただきます。
高校入試対策の革新的ソリューションをご提案します。



===== スライド2：今日お話しすること =====
本日は4つのポイントでお話しします。
まず塾の現場で起きている問題、次にAIで解決できること、
そして実際に作ってみた結果、最後にビジネスとしての可能性についてです。



===== スライド3：塾講師の悩み =====
現在、塾講師の皆さんが抱えている大きな悩みが3つあります。
1つ目は過去問が足りないこと。販売されているのは約10年分だけです。
2つ目は生徒がやり尽くしてしまうこと。新しい問題がもうありません。
3つ目は問題作成に時間がかかること。質の高い問題を作るのは本当に大変です。



===== スライド4：もしも... =====
もしも、無限に問題が作れたらどうでしょうか？
そんな夢のような話が、AIの力で実現できるかもしれません。
AIが問題を自動生成してくれるシステムです。



===== スライド5：解決策：AI問題生成システム =====
具体的な解決策をご紹介します。
まず過去問を学習させ、各高校の傾向を分析します。
そして瞬時に問題生成し、必要な分だけ作成可能です。
さらにPDF出力に対応し、紙に印刷してすぐ使えます。



===== スライド6：実際に試してみました =====
理論だけでは説得力がありませんので、
実際にChatGPTで数学問題を生成してみました。
その結果をお見せします。



===== スライド7：実験結果①：微分問題 =====
こちらが実際に生成された微分問題です。
ご覧いただけるように、高校レベルの適切な問題が生成されました。
数学的にも正確で、実際の授業で使えるレベルです。



===== スライド8：実験結果②：解答付き問題 =====
さらに素晴らしいのは、解答付きの問題も生成できることです。
詳しい解答プロセスも自動生成されます。
これにより、講師の負担が大幅に軽減されます。



===== スライド9：実験結果③：演習問題 =====
こちらは演習問題の例です。
実際の授業で使えるレベルの問題が生成されています。
様々な問題パターンに対応可能です。



===== スライド10：実験結果④：増減表問題 =====
より複雑な問題形式にも対応しています。
こちらは増減表を使った問題です。
AIの問題生成能力の幅広さがお分かりいただけると思います。



===== スライド11：実験で分かったこと =====
これらの実験により、技術的に実現可能であることが証明されました。
数学的に正確な問題が生成でき、
解答プロセスも論理的で、高校入試レベルに適合しています。



===== スライド12：誰に売る？ =====
ターゲットは塾講師の皆さんです。
NSG教育研究会、森塾、TOP、Polaris長岡など、
地域の主要な学習塾に展開していきます。



===== スライド13：どうやって稼ぐ？ =====
収益モデルは3つです。
月額サービスとして塾向けの問題生成サービスを提供し、
問題ストック機能で生成した問題を蓄積・再利用し、
最終的には蓄積した問題で教材販売も行います。



===== スライド14：課題：お金がかかる =====
もちろん課題もあります。API料金が高いことです。
しかし、解決策があります。
問題を貯める仕組みを作り、一度作った問題は再利用することで、
コストを大幅に削減できます。



===== スライド15：課題：図が難しい =====
もう一つの課題は図形問題の精度です。
これも解決策があります。
人間と協力するハイブリッドアプローチです。
AIが文章を生成し、人間が図を作成します。



===== スライド16：まずは数学から =====
最初は数学から始めます。
数学が一番需要がありそうな科目だからです。
図は講師が描けますし、問題パターンも豊富です。



===== スライド17：期待される効果 =====
このシステムにより、大きな効果が期待されます。
塾講師にとっては時間短縮と無制限の問題供給が実現し、
生徒にとっては豊富な練習機会と入試対策強化が可能になります。



===== スライド18：今後の計画 =====
今後の計画は3段階です。
まず数学プロトタイプを完成させ、
次に塾での実証実験を行い、
最後に他科目への展開を進めます。



===== スライド19：成功の鍵 =====
成功の鍵は「塾講師×AIエンジニア」という組み合わせです。
現場を知っている経験と技術力、
そして地域に根ざした信頼関係が私たちの強みです。
